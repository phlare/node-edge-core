# node-edge-core

Node Edge Core is a reusable TypeScript service template for integration-facing services.

It is intended for things like:

- Slack adapters
- MCP servers
- webhook receivers
- lightweight integration APIs

This repo provides the generic service plumbing needed to build those systems quickly and consistently.
